Cloud Savedata Manager Documentation

About

Cloud Savedata Manager is a tool that allows the user to backup & restore their game savedata making use of their own Google Drive account. Even though this tool is designed for the games that don't allow native cloud saves on their respective launchers, it is possible to backup those savedata aswell.

Initial Setup


Google Drive Credential

The first step is to obtain a Google Drive API Credential

  • Head to this link and click on Enable the Drive API
  • Enter any name for the project
  • Select Yes and Next
  • Select Desktop app and Create
  • Click Download Client Configuration
  • Place the credentials.json inside the credentials folder (make sure that the json file has that exact name)
  • On the console, press enter to continue

Login on Google Drive

The second step is to obtain a Google Drive API Credential. A new web browser will open asking you to login to your Google Account

  • Login on Google Drive
  • Click on Advanced Configuration and then on Quickstart
  • Allow the required permissions

Now you should be able to make use of the Cloud Savedata Manager with your own Google Drive account. Click the button bellow to start using the app.
